Detailed Description

Provides implementations for managing and accessing periodic table data.

Implements functions to load a periodic table from a file, sort it by atomic number, and retrieve the atomic number of an element given its chemical symbol.

Dependencies:

  • periodicTable.h: Provides support for element data management.
