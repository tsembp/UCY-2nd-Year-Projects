Detailed Description

Implementation of a Latin Square game.

This file contains the core functionality for handling a Latin Square game. It provides options to load a Latin Square from a file, display it, accept user inputs to modify the square, and save the final state to a file.

Checks if the user input is valid for the Latin Square.

This function validates the user input (i, j, val) for the Latin Square based on several criteria:

  • Indices must be within bounds.
  • Values should not be inserted in pre-given cells.
  • A value cannot appear more than 'size' times in the square.
  • Values cannot be repeated in the same row or column.
